Hari Dandapani's Computational Biology Website

This website is a fun exploratory project for me to test out a lot of the algorithms that we learned in my Computational Biology class at Brown University and to implement new algorithms that were briefly introduced in the class. I had a lot of fun making this site, and I hope that you have a lot of fun too!

The alignment page allows you to test out different alignment algorithms including local alignment, global alignment, global alignment with affine gap penalty, and global alignment with logarithmic gap penalty with different settings.

Alignment

The Deterministic Finite Automata tool makes a DFA for a given string.

Deterministic Finite Automata

The UPGMA Tool makes a UPGMA for a given phylogeny.

UPGMA

The Neighbor Joining Tool makes a Neighbor Joining graph for a phylogeny

Neighbor Joining

The KMP tool runs the Knuth Morris Pratt algorithm and finds the failure function.

KMP

The Burrows-Wheeler Transform lets you try out the block-sorting compression in both the forward and backward direction.

Burrows-Wheeler Transform
