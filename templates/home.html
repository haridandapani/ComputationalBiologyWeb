Hari Dandapani's Computational Biology Website

This is a fun exploratory project for me to test out a lot of the algorithms that we learned in my Computational Biology class at Brown University and to implement new algorithms that were briefly introduced in the class.

The alignment page allows you to test out different alignment algorithms with different settings.

Alignment

The Deterministic Finite Automata tool makes a DFA for a given string.

Deterministic Finite Automata

The UPGMA Tool makes a UPGMA tree for a given phylogeny.

UPGMA

Make a suffix tree for any string.

Suffix Tree

The Neighbor Joining Tool makes a Neighbor Joining graph for a phylogeny

Neighbor Joining

The KMP tool runs the Knuth Morris Pratt algorithm and finds the failure function.

KMP

The Burrows-Wheeler Transform lets you try out the block-sorting compression in both the forward and inverse direction.

Burrows-Wheeler Transform
